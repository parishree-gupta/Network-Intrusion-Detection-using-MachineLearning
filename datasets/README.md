# Dataset

This project uses the NSL-KDD dataset.

Download the dataset from one of the following sources:

- https://www.unb.ca/cic/datasets/nsl.html
- https://www.kaggle.com/datasets/hassan06/nslkdd

Place the downloaded dataset files in the `datasets/` folder.

The files `bin_data.csv` and `multi_data.csv` are generated automatically by the preprocessing notebook and are therefore not included in this repository.
